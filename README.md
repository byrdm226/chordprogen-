# ChordProGen

ChordProGen is a simple browser-based tool that generates chord progressions for a given song key.  
It helps musicians and songwriters quickly find compatible chords for their songs.

## Features
- Generate chord progressions in any key
- Clean, responsive, mobile-friendly interface
- No installation required â€” runs in any modern browser

## How to Use
1. Open ChordProGen in your browser.
2. Select the desired song key.
3. Click "Generate" to get a chord progression.
4. Copy or note the generated chords for your song.

## Deployment
This tool is hosted on [Netlify](https://chordprogen.netlify.app).
